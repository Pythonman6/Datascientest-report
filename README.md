### Objectif
L'objectif de ce projet est de créer une application web Streamlit capable de prédire la survie ou non d'un passager du Titanic.\
Pour cela, je travaillerais sur le fichier train.csv téléchargeable sur le lien : https://www.kaggle.com/competitions/titanic/data?select=train.csv.\
Nous aurons affaire à un problème de classification binaire.
### Définition
Classification : tâche qui consiste à prédire une catégorie (classe) à partir de données.\
Binaire : il n’y a que deux classes possibles. Dans notre cas, survie (1) ou non(0).
